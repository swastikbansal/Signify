# ISL to Speech - Sign Language Guide

This guide provides basic information about the Indian Sign Language (ISL) gestures supported by the system. It's intended to help users understand how to perform the signs recognized by the application.

## Basic Signs

### Pronouns

| Sign   | Description                                       |
| ------ | ------------------------------------------------- |
| **I**  | Point index finger toward yourself at chest level |
| **My** | Place open hand (palm facing in) against chest    |

### Common Actions

| Sign        | Description                                                                          |
| ----------- | ------------------------------------------------------------------------------------ |
| **Eat**     | Bring cupped hand to mouth repeatedly                                                |
| **Go**      | Extend arm forward with index and middle finger extended, moving away from body      |
| **Help**    | Place one flat hand on top of the other palm-up hand and lift upward                 |
| **Support** | One fist supporting the other arm from below                                         |
| **Dance**   | Both hands with index and middle fingers extended, alternating up and down movements |

### Objects

| Sign      | Description                                                     |
| --------- | --------------------------------------------------------------- |
| **Sun**   | Form a circle with both hands above head                        |
| **Water** | Form 'W' hand shape near mouth                                  |
| **Paper** | Flat hands moving apart as if opening a sheet of paper          |
| **Book**  | Open and close palms together, like opening and closing a book  |
| **Pizza** | Draw a circle with index fingers and then make a cutting motion |

### Places

| Sign       | Description                          |
| ---------- | ------------------------------------ |
| **School** | Clap hands twice and then cross arms |

### People

| Sign        | Description                               |
| ----------- | ----------------------------------------- |
| **Teacher** | "T" handshape moved from forehead outward |
| **Friend**  | Hook index fingers together               |
| **Woman**   | Thumb traced along jawline                |
| **Deaf**    | Point to ear and then to mouth            |
| **Winner**  | Raise both fists in celebration           |

### Emotions/States

| Sign      | Description                                         |
| --------- | --------------------------------------------------- |
| **Happy** | Circular motion with flat hands in front of face    |
| **Sad**   | Drag fingers downward from eyes, representing tears |
| **Love**  | Cross arms over chest                               |

### Descriptors

| Sign      | Description                                 |
| --------- | ------------------------------------------- |
| **Deep**  | One hand moving downward, palm facing down  |
| **Flat**  | Flat hand moving horizontally               |
| **Thick** | Show thickness using thumb and index finger |
| **High**  | Raise hand high above head                  |
| **Loud**  | Open hand moving outward from mouth         |
| **Slow**  | One hand slowly moving forward              |
| **Soft**  | Gentle touching motion with fingertips      |
| **Quiet** | Index finger placed over lips               |
| **Poor**  | Empty pockets motion                        |

### Other Concepts

| Sign          | Description                                 |
| ------------- | ------------------------------------------- |
| **Important** | Index finger pointing upward and emphasized |
| **Accident**  | Two fists colliding                         |
| **Yes**       | Head nod or fist nodding up and down        |
| **ISL**       | Finger-spell "I-S-L"                        |

## Tips for Best Recognition

1. **Clear Gestures**: Make sure your gestures are clear and deliberate
2. **Proper Positioning**: Position yourself so your entire upper body is visible
3. **Good Lighting**: Ensure your environment is well-lit
4. **Contrast**: Wear clothing that contrasts with your skin tone and background
5. **Face the Camera**: Orient your body toward the camera
6. **Appropriate Distance**: Stand about 3-4 feet from the camera
7. **Steady Pace**: Perform signs at a moderate, steady pace
8. **Complete Motions**: Finish each sign completely before starting the next
